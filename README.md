# Prueba tecnica: Security and System

## Como levantar todo el proyecto
1. Clonar el repositorio
```
git clone git@github.com:FededelRincon/Security-and-System-Tecnical.git
```

2. instalar dependecias utilizando su gestor de dependencias
```
npm install
yarn install
pnpm install
```

4. correr el proyecto
```
npm run dev
```
y desde el navegador acceder a alguna de las url
```
http://localhost:3000/
http://localhost:3000/profile
```

5. correr tests
```
npm run test
```
